# Projeto final Tech Talents
## Web site de organização financeira

Projeto desenvolvido no *hackathon* promovido pela **Neon** para finalizar o programa *Tech Talents*, tendo como resultado um app para lançamentos de organização financeira.

### Ficha tecnica
- Biblioteca: BootStrap5, Poppers.js, google fonts(Roboto).
#### Tecnologias
- HTML5,
- CSS3,
- Javascript,
- Java8

## Recursos que adicionamos

- Um popup personalizado ao preencher o formulario.
- Poder editar a transação.
- Salvar o lançamento do usuario no localStorage, para que apareçam mesmo que a página for carregado.
- Colocar uma cor de acordo com a carteira: vermelho para saúde negativa e verde para saudável
